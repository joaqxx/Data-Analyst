Business Problem

Managers needed a concise view of user tasks and focus habits to improve productivity coaching and feature prioritization. Without visibility into task completion patterns and time allocation, decision-makers couldn't identify optimization opportunities.

Key Questions

1

Which tasks take the longest on average?

2

How does completion rate vary by weekday?

3

Which features drive the most repeat usage?

Analysis & Findings

FINDING 1

Task Duration Distribution: Email tasks average 28 minutes, followed by meetings (22 min) and admin work (18 min). Top 10 tasks represent 60% of total time spent.

FINDING 2

Weekly Patterns: Completion rates peak Tuesday-Thursday (72%) and drop significantly on Mondays (45%) and Fridays (52%). Weekend completion falls to 28%.

FINDING 3

Feature Usage: Recurring tasks have 3.2x higher engagement. Push notifications boost completion by ±28%. Users most active between 6-9 PM.

Approach & Methodology

1

Data Collection

Exported user activity logs and comprehensive task metadata from the database.

2

Data Cleaning & Preparation

Removed duplicates, standardized timestamps, and handled missing values using Power Query.

3

Analysis & KPI Calculation

Used PivotTables to calculate average time, completion rates, and user engagement metrics.

4

Dashboard Development

Built interactive visualizations with filters and drill-down capabilities for stakeholder insights.
